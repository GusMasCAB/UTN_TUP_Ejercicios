HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web mediante el uso de etiquetas y atributos. Con HTML, los desarrolladores pueden crear documentos que incluyen texto, imágenes, enlaces, formularios y otros elementos interactivos.

HTML es fundamental para la World Wide Web, ya que proporciona la estructura básica de cualquier página web. Junto con CSS (Cascading Style Sheets) y JavaScript, HTML forma la base de la creación y el diseño de sitios web modernos y dinámicos.
